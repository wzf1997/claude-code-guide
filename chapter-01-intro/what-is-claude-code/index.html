Claude Code 是什么？它能做什么？

🟢 入门

本文介绍 Claude Code 的基本概念、核心能力和典型使用场景。

什么是 Claude Code？​

Claude Code 是 Anthropic 出品的 AI 编程助手，以命令行工具形式运行在你的本地环境。 与 ChatGPT 等网页聊天不同，Claude Code 能够直接操作你的文件系统、执行终端命令、搜索代码库。

它能做什么？​

  • 阅读、修改、创建任意文件
  • 执行 Bash 命令（运行测试、安装依赖等）
  • 理解整个代码库的结构
  • 使用工具（Tool）链式完成复杂任务
